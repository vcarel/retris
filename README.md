# Retris
A React Tetris clone

[https://vcarel.github.io/retris/](https://vcarel.github.io/retris/)

## Getting started
Checkout the project, then:
```
yarn install
yarn start
```

To build the app:
```
  yarn build
```

To serve the built locally:
```
  yarn serve
```

 _Have fun!_
